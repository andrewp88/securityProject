Installation

Please follow these simple steps to install the plugin and import the demo content in just few clicks.

  1. Go to the WP admin panel › Plugins
  2. Click the Add New button


  1. Click the Upload Plugin button
  2. Find the snax.zip file inside the package you've downloaded from CodeCanyon
  3. Upload it


  1. After few seconds the plugin will be installed.
  2. Click the Activate Plugin button
