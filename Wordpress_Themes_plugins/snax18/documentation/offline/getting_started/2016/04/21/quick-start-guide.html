Quick-start guide

For the impatient, here’s how to get the plugin up and running in a second.

  1. Go to the WP admin › Plugins › Add New, upload snax.zip and activate it. Need more details?
  2. Under the WP admin › Settings › Snax › Pages, click the View button next to the Frontend Submission page.
  3. Choose what kind of story you want to create (eg. Create open list).
  4. Upload an image or paste embed URL, fill in the Title and accept the Terms and Conditions.
  5. Click the Publish button and voilà, you've just submitted your first post!
