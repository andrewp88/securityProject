Demo Mode

The demo mode is the fastest way to start playing with frontend submission. Thanks to it, you don't need to upload new images/embeds all the time you want to test submission process.

Setup

To enable demo mode, please do as follows:

  1. Go to the WP admin panel › Snax.
  2. Switch to the Demo section.
  3. Check the Demo mode enabled? box.
  4. From the Media Library choose (or upload new ones) images that will be used as your demo images.
  5. Fill in embed URL and choose post image if you want to test embeds.
  6. Now, when you create a new post, demo data will show up under the upload image/embeb form.
  7. Save changes.
