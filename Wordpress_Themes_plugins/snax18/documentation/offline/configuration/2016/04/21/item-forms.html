Item Forms

Item forms are types of content users can add to existing lists. By default, these are New Image and New Embed forms. If you want to have a list consists of only images, you can disable the Embed form and that's all.

Global Setup

To configure forms globally, please do as follows:

  1. Go to the WP admin panel › Snax.
  2. Switch to the Lists tab.
  3. Uncheck all forms you don't want to use.
  4. Save changes.

Single List Setup

If you want to override global setup, just for one or few lists, please follow these steps:

  1. Go to the WP admin panel › Posts.
  2. Find your list (using filters and search form).
  3. Go to its edit page.
  4. In the Snax metabox, check the "Change Default Forms" field.
  5. Uncheck all forms you don't want to use.
  6. Save changes.
